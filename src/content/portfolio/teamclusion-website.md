- Logodesign und Corporate Identity

- Responsive Website mit Bootstrap und Jekyll
- Deployment auf eigenem Server

- **HTML/CSS** – Struktur und Styling
- **Bootstrap** – Responsive Framework
- **Jekyll** – Static Site Generator
